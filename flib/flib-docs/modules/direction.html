# to_vector_2d(direction, longitudinal, orthogonal)

Returns a two-dimensional vector from a cardinal direction.

Only supports cardinal (four-way) directions.

Parameters:
  • direction : (defines.direction)
  • longitudinal : (number) Distance to move in the specified direction.
  • orthogonal : (number) Distance to move perpendicular to the specified direction. A negative distance will move “left” and a positive distance will move “right” from the perspective of the direction.
Returns:
  • (Position)
